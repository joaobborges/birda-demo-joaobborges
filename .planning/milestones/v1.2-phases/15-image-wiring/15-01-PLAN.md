  <action>
Create `scripts/download-images.js` — a Node.js script that:
1. Reads the Wikipedia image URLs from `src/data/birds.ts` by parsing the file with a regex (extract each bird's name and image URL from the raw text)
2. For each bird, converts the name to kebab-case (e.g., "European Robin" -> "european-robin")
3. Downloads the image from the Wikipedia URL and saves as `src/assets/birds/{kebab-name}.jpg`
4. Creates the `src/assets/birds/` directory if it doesn't exist
5. Logs progress for each download (e.g., "Downloaded: european-robin.jpg")
6. Handles errors gracefully (log and continue if one fails)

Use Node built-ins only (https, fs, path). No external dependencies needed — the Wikipedia URLs return JPEG content directly when fetched with a proper User-Agent header (set User-Agent to "BirdaApp/1.0" to avoid 403s from Wikipedia).

IMPORTANT: Wikipedia thumbnail URLs redirect. Use `https` module and follow redirects manually (check for 301/302 response.statusCode, read the `location` header, make a new request). Or use `fetch` if available in the project's Node version.

Add to package.json scripts: `"download-images": "node scripts/download-images.js"`

Then RUN the script to actually download all 20 images.
