  <action>
Create `src/components/map/CaptureFAB.tsx` — a self-contained FAB component with animated speed-dial menu.

**FAB button:**
- 56px circle, backgroundColor `semantic.actionPrimary`, centered white icon
- Positioned absolutely: `bottom: 24, right: 16` (will sit above tab bar when placed in Map screen)
- Icon: Ionicons `add` (28px, white). When menu open, icon rotates 45 degrees to form an X shape (use `withSpring` on a `useSharedValue` controlling rotation)
- Add subtle shadow: `boxShadow: '0 4px 12px rgba(0, 0, 0, 0.25)'`
- Use `borderCurve: 'continuous'` on the circle

**Menu items (3 options, fanning upward from FAB):**
- Camera: Ionicons `camera` icon
- Microphone: Ionicons `mic` icon
- Notes: Ionicons `document-text` icon
- Each item: 44px circle, white background, `semantic.actionPrimary` icon color, with a text label to the right
- Items stack vertically above the FAB with 12px spacing between them
- Right-aligned with the FAB (same `right` position)
- Text labels: `typography.caption` style, positioned to the left of each circle

**Animation (all Reanimated native thread):**
- Track open/closed state with a `useSharedValue` (0 = closed, 1 = open)
- FAB press toggles state: `withSpring(isOpen ? 0 : 1, { damping: 15, stiffness: 200 })`
- FAB icon rotation: interpolate shared value 0->1 to 0deg->45deg
- Menu items: each item translates from Y=0 (at FAB position) to its final Y offset, with scale 0->1. Use staggered delays: item 1 (Notes, closest) 0ms, item 2 (Mic) 30ms, item 3 (Camera, farthest) 60ms
- Each menu item also fades in: opacity 0->1

**Overlay:**
- When menu open, render a full-screen `Pressable` behind the menu items but above the map
- Semi-transparent dark: `backgroundColor: 'rgba(0, 0, 0, 0.3)'`
- Animated opacity synced with the open/close shared value
- `onPress` closes the menu
- Use `StyleSheet.absoluteFillObject` for positioning

**Component API:**
```typescript
export function CaptureFAB(): JSX.Element
```
No props needed — this is visual-only, menu items are non-functional (per requirements: "no functional integration required"). Each menu item press should just close the menu for now.

Use `StyleSheet.create()` for all styles. Use `Pressable` from react-native (not TouchableOpacity). Follow project pattern: ternary for conditional rendering, never `&&`.
