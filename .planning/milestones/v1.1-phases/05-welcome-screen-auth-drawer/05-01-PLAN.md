2. Create `src/components/welcome/BirdMosaic.tsx` -- an animated 3-column Pinterest-style mosaic of bird images. Implementation details:

**Props interface:**
```typescript
interface BirdMosaicProps {
  height: number  // container height (caller passes ~55% of screen)
}
```

**Column data setup:**
- Import `birds` from `@/data/birds` and use their `.image` URLs
- Split 20 birds across 3 columns (indices 0-6, 7-13, 14-19 or similar)
- Pre-define tile heights using a deterministic array: `[140, 180, 160, 200, 150, 170, 190, 145, 175, 165]` cycled per tile index. Do NOT use actual image dimensions.
- Each tile has `borderRadius: 12` with `borderCurve: 'continuous'` and ~4px gap between tiles

**Animation:**
- Use `useSharedValue` + `useAnimatedStyle` + `withRepeat(withTiming(...), -1, false)` for each column
- Column directions: left=up (translateY goes negative), center=down (positive), right=up (negative)
- Speeds: left=25000ms, center=30000ms, right=22000ms per cycle (Claude's discretion from RESEARCH.md)
- Use `Easing.linear` for constant-speed scrolling
- Start animation in a `useEffect` on mount

**Seamless loop:**
- Render each column's image list TWICE (stacked vertically) so the animation loops seamlessly
- Calculate `totalHeight` as sum of all tile heights + gaps for one set
- translateY target = -totalHeight for upward columns, +totalHeight needs offset (start at -totalHeight, animate to 0) for downward column

**Image rendering:**
- Use `Image` from `expo-image` with `contentFit="cover"` and `priority="low"` (decorative)
- Add a placeholder background color (semantic.bgTinted) so blank tiles look okay while loading

**Container:**
- `overflow: 'hidden'` to hard-crop at top and bottom edges (no gradient fade per user decision)
- `pointerEvents="none"` on the outer container to prevent scroll interaction (WELC-01 partial)
- Use `flexDirection: 'row'` with `gap: 4` for the 3-column layout

**Conventions:**
- Named export: `export function BirdMosaic(...)`
- `StyleSheet.create()` for styles
- `@/` alias for imports
- `borderCurve: 'continuous'` on all rounded corners
- Ternary with `null` for conditional rendering (never `&&`)
